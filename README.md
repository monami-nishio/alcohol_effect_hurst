# Alcohol impacts an fMRI marker of neural inhibition in humans and rodents

## About
This is the codes to replicate the analysis in *"Alcohol impacts an fMRI marker of neural inhibition in humans and rodents"*.  

**To reproduce Figure 1**  
- `rat/Fig1.ipynb` — Calculates and plots the region-wise T1w/T2w distribution for both macaques and humans.  

**To reproduce Figure 2**  
- `human/Fig2,3.Rmd` — Calculates and plots the developmental trajectory of T1w/T2w in macaques and humans. Also contains the subsampling code to replicate Figure S5.  

**To reproduce Supplementary Figure 1**  
- `rat/FigS1.ipynb` — Plots the correlation between geodesic distance and the sensory-association axis for both macaques and humans.  

**To reproduce Supplementary Figure 2**  
- `human/FigS2.ipynb` — Calculates and plots the region-wise MBP distribution in macaques.
